# Exp-automl-projeto-01
Projeto oi do bootcamp da DIO
Treinando Seu Primeiro Modelo de Machine Learning Para Prever Vendas

Baseado no conceito e na ferramenta adaptei meu projeto para prever os melhores produtos consumidos em um app de delivre por exemplo.
hoje atuo como gestor de projetos e faço pesquisas de demanda e viabilidade para iniciar ovos negócios
Em um senário hipotético imagine que consegui uma base de dados de alguns aplicativos de delivre ou até de empresas locais 
onde adequei os dados de forma que estejam em conformidade com a análise do agente, pedi para que ele pudesse prever qual o melhor produto
com maior demanda que resultasse em uma oportunidade de implantação.

após a criação e configuração dos datasets foi criado o ambiente e as ferramentas de processamento para rodar o pipeline
